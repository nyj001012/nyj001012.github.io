<h4>
<pre>
문자열 객체
- 여러 메서드 존재
- 자연어(사람이 사용하는 언어)<br>
문자열 포멧팅
- 원하는 스타일로 문자열을 만드는 것
- 포멧 지정: "문자열 %지시어" % ("대입 문자열", ...)<br>
지시어
- 자료형에 따라 다른 지시어를 지정
- %s: 문자열
- %c: 문자 1개
- %d: 정수
- %f: 실수(%.소수점f)
- %o: 8진수
- %x: 10진수 
- %%: literal(문자 '%')<br>
이스케이프 문자(\)
- 문자열을 만들 때 고유한 의미를 갖는 문자
- \b: 백스페이스
- \t: 탭
- \n: 줄 바꿈
- \f: 다음 페이지 출력
- \r: 시작 라인으로 바꿈
- \\: 역슬래시
- \': 작은 따옴표
- \": 큰 따옴표
</pre>
