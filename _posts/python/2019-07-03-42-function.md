함수
- 코드의 논리적인 흐름을 묶는 역할
- 기본 인자: def 함수명(인자, default=값):
              실행코드
            return 반환값;
- 가변 인자: def 함수명(인자, *인자):
              실행코드
            return 반환값;
            반복문 필요
- 별칭 인자: def 함수명(별칭):
              실행코드(별칭 사용)
            return 반환값;
- 다중 반환 값: def 함수명(별칭):
                 실행코드
               return 반환값, 반환값, ...;
</pre>
