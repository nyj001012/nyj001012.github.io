<h4>
<pre>
모듈
- python 코드를 하나로 묶어서 사용(하나의 *.py)
- import 모듈명
- from 모듈명 import 가져올 함수, ...
- from 모듈명 import 가져올 함수 as 앞으로의 이름<br>
모듈 찾는 순서
- 현재 directory
- 환경변수에 지정된 경로
- python 라이브러리 경로<br>
패키지
- 모듈들의 집합
- from 패키지명 import 모듈명
- from 패키지명 import 모듈명 as 앞으로의 이름<br>
