CSS(Cascading Style Sheets)
- &lt;style&gt;css 코드&lt;/style&gt;: HTML 스타일을 적용하기 위한 코드 작성을 명시
- 엘리먼트 단위로 지정
- 웹 페이지 단위로 지정
- 웹 사이트 단위로 지정
- 엘리먼트 {
    속성: 값;
  }<br>
식별자
- id: 하나의 페이지의 유일한 요소의 식별자 역할, 엘리먼트 이름 대신 #id
- class: 하나의 페이지의 다중 요소의 식별자 역할, 엘리먼트 이름 대신 .class
