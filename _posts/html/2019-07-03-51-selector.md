CSS 선택자
- 엘리먼트의 관계를 이용해서 선택하는 방법<br>
자손 선택자
- 선택자 선택자 {
    CSS 코드
  }<br>
자식 선택자
- 선택자 > 선택자 {
    CSS 코드
  }<br>
인접 형제 선택자
- 선택자 + 선택자 {
    CSS 코드
  }<br>
일반 형제 선택자
- 선택자 ~ 선택자 {
    CSS 코드
  }
