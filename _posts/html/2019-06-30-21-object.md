객체
- 자바 스크립트 프로그램의 대상
- 속성 + 메서드로 이루어짐
- var 객체명 = {키: 값, 키: 값, ...};
  키: 속성(Property)<br>
객체 사용
- 객체.메서드();: 메서드를 실행
- 객체.프로퍼티;: 속성을 가져옴(get)
- 객체.프로퍼티 = 값;: 속성을 변경(set)<br>
내장 객체
- 자바 스크립트가 기본적으로 제공하는 객체
- object, number, date, math, dom, bom, ...<br>
Math 객체
- 수학 관련 정보를 얻어올 수 있음
