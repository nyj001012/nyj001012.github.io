JAVA SCRIPT
- 사용자의 동작(이벤트)에 반응(핸들링)하기 위한 언어<br>
이벤트
- 웹 페이지에서 발생하는 모든 사용자의 액션
- 이벤트 핸들러는 이벤트 이름 앞에 "on"이 붙음<br>
이벤트 핸들러 등록 방법
- 속성: 속성으로 등록
- HTML DOM: DOM 객체 탐색으로 등록
- addEventListener(): 메서드를 통해 이벤트 중복해서 등록
