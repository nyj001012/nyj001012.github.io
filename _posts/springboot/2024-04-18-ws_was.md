# 웹 서버 (Web Server)
- HTTP 기반
- **정적 리소스 제공**, 기타 부가기능
- 예) NGINX, APACHE

# 웹 애플리케이션 서버 (WAS; Web Application Server)
- HTTP 기반
- 웹 서버 기능 + 애플리케이션 로직 수행
  - 동적 HTML, HTTP API
  - 서블릿, JSP, 스프링 MVC
- **애플리케이션 코드를 실행**하는 데 특화
- 자바는 서블릿 컨테이너 기능을 제공하면 WAS
- 예) 톰캣(Tomcat), Jetty, Undertow
