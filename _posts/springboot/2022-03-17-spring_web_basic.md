# MVC와 템플릿 엔진
MVC = Model + View + Controller  
템플릿 엔진은 HTML 파일을 그냥 브라우저에 전달하는 것이 아니라, 서버에서 별도의 동작을 한 후에 데이터를 동적으로 변환하여 전달한다.

# API
API는 json과 같은 포맷으로 클라이언트에 전달하는 방식이다. 주로 서버 - 서버가 데이터를 주고 받을 때 사용한다.
